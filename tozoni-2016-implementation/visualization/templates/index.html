AGSol instance viewer

Paste a one-line .pol file (see README §4.a) or use the CLI below. Optional .sol (§4.c).

Default solver path: {{ solver_default }} — set AGSOLVER to override.

CLI

# from repository root (966/)
python3 -m venv tozoni_venv
tozoni_venv/bin/pip install -r tozoni-2016-implementation/visualization/requirements.txt
tozoni_venv/bin/python tozoni-2016-implementation/visualization/agp_viz_app.py /path/to/instance.pol -o out.png
# or: visualization/run_viz.sh /path/to/instance.pol -o out.png
# web UI (no args)
tozoni_venv/bin/python tozoni-2016-implementation/visualization/agp_viz_app.py
